# Fantasy Bank

JavaScript practice project focused on application logic, DOM manipulation, and basic authentication flow.

This project currently includes login and account registration. The dashboard and account features are under development and will be added in future updates.

---

## Features

- Login system
- Account registration
- View switching between login and register
- Random 7-digit account number generation
- Password validation
- In-memory session logic

---

## Technologies

- JavaScript (ES6)
- HTML5
- CSS3

---

## In Development

- User dashboard
- Balance visualization
- Deposit and withdrawal simulation
- Database integration (future)
